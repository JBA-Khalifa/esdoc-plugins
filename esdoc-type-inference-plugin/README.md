# ESDoc Type Inference Plugin(PoC)
**This is Proof of Concept**
## Install
```bash
npm install esdoc-type-inference-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-type-inference-plugin", "option": {"enable": true}}
  ]
}
```

- `enable` is default `true`
