# ESDoc Exclude Source Plugin
## Install
```bash
npm install esdoc-exclude-source-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-exclude-source-plugin"}
  ]
}
```
