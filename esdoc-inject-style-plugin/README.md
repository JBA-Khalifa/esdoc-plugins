# ESDoc Inject Style Plugin
## Install
```bash
npm install esdocs-inject-style-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdocs-inject-style-plugin", "option": {"enable": true, "styles": ["./foo.css"]}}
  ]
}
```

`enable` is default `true`.
