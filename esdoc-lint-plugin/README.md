# ESDoc Lint Plugin
## Install
```bash
npm install esdocs-lint-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdocs-lint-plugin", "option": {"enable": true}}
  ]
}
```

`enable` is default `true`
