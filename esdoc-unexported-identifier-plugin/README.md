# ESDoc Unexported Identifier Plugin
## Install
```bash
npm install esdocs-unexported-identifier-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdocs-unexported-identifier-plugin", "option": {"enable": false}}
  ]
}
```

`enable` is default `false`.
