# ESDoc JSX Plugin

## Install

```bash
npm install esdocs-jsx-plugin
```

## Config

```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdocs-jsx-plugin", "option": {"enable": true}}
  ]
}
```

- `enable` is default `true`
