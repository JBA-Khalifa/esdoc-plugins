# ESDoc Inject GTM Plugin

## Install

```bash
npm install esdoc-inject-gtm-plugin
```

## Config

```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-inject-script-plugin", "option": {"enable": true, "id": "GTM-XXXXX"}}
  ]
}
```

`enable` is default `true`.
